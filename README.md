# Local Area Driver Booking System

This is a web-based application to book local drivers easily.

## Tech Stack
* **Frontend:** HTML, CSS, JS
* **Backend:** PHP
* **Database:** MySQL

## Features
* User/Driver Registration and Login
* Real-time Driver Booking
* Admin Dashboard to manage users

## Setup Instructions
1. Clone the repository into your XAMPP `htdocs` folder.
2. Import the `database.sql` file into your phpMyAdmin.
3. Open `http://localhost:8080/driver_booking/index.php` in your browser.
